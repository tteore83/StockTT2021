Définition

L'intégration est la transposition du web design en code informatique. Concrètement, le développeur front-end reprend la maquette validée du site et la transforme en langage HTML, CSS et JavaScript via un éditeur de texte. Chaque choix graphique du designer sera transformé en ligne de code. On pourrait utiliser la métaphore suivante : le designer parle en français à un ordinateur, or cet ordinateur ne comprend que l'anglais, le développeur sera donc en charge de traduire le message du designer en anglais pour que l'ordinateur puisse le comprendre. On parle donc bien de langage !
